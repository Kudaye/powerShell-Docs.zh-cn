<span data-ttu-id="332bf-197">按照以下步骤创建调试配置文件：</span><span class="sxs-lookup"><span data-stu-id="332bf-197">Follow these steps to create your debug configuration file:</span></span>

  1. <span data-ttu-id="332bf-198">通过按 <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>D</kbd>，在 Windows 或 Linux 上打开“调试”  视图。</span><span class="sxs-lookup"><span data-stu-id="332bf-198">Open the **Debug** view on Windows or Linux by pressing <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>D</kbd>.</span></span> <span data-ttu-id="332bf-199">在 macOS 上，按 <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>D</kbd>。</span><span class="sxs-lookup"><span data-stu-id="332bf-199">On macOS, press <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>D</kbd>.</span></span>
  1. <span data-ttu-id="332bf-200">单击工具栏中的“配置”齿轮图标  。</span><span class="sxs-lookup"><span data-stu-id="332bf-200">Click the **Configure** gear icon in the toolbar.</span></span>
  1. <span data-ttu-id="332bf-201">VSCode 将提示“选择环境”  。</span><span class="sxs-lookup"><span data-stu-id="332bf-201">VSCode prompts you to **Select Environment**.</span></span> <span data-ttu-id="332bf-202">选择“PowerShell”  。</span><span class="sxs-lookup"><span data-stu-id="332bf-202">Choose **PowerShell**.</span></span>
