```Syntax

{

    [ Credential = [PSCredential] ]
    [ Description = [string[]] ]
```

```powershell
    MembersToInclude= "$domain\$($Node.AdminAccount)"
```

## <a name="example-3"></a><span data-ttu-id="e255f-153">示例 3</span><span class="sxs-lookup"><span data-stu-id="e255f-153">Example 3</span></span>

<span data-ttu-id="e255f-154">下面的示例展示了如何确保服务器 TigerTeamSource.Contoso.Com 上的本地组 TigerTeamAdmins 不包含特定域帐户 Contoso\JerryG。</span><span class="sxs-lookup"><span data-stu-id="e255f-154">The following example shows how to ensure a local group, TigerTeamAdmins, on the server TigerTeamSource.Contoso.Com does not contain a particular domain account, Contoso\JerryG.</span></span>

```powershell
Configuration SecureTigerTeamSource {
    Import-DscResource -ModuleName 'PSDesiredStateConfiguration'

    Node TigerTeamSource.Contoso.Com {
        Group TigerTeamAdmins {
            GroupName        = 'TigerTeamAdmins'
            Ensure           = 'Present'
            MembersToExclude = "Contoso\JerryG"
        }
    }
}
```
