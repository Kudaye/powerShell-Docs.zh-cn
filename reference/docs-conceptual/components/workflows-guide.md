# <a name="psworkflow-module"></a>PSWorkflow 模块

本部分包含有关 **PSWorkflow** 模块的帮助主题，此模块包含支持 Windows PowerShell 工作流功能的 cmdlet。 PSWorkflow  模块随 Windows PowerShell 5.1 一起安装。

> [!NOTE]
> 只有 Windows PowerShell 5.1 支持工作流。
